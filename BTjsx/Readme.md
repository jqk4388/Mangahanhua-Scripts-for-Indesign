# InDesign 漫畫輔助定位腳本集

這套腳本主要用於漫畫排版作業，能夠透過讀取外部 JSON 的座標資料，在 InDesign 中自動生成對話框的輔助定位方框，並提供一鍵讓文字框自動吸附到輔助框中心的功能。

## 1. 創建方框.jsx

**用途：** 
讀取包含對話框座標資訊（`xyxy_pixel`）的 JSON 檔案，並在 InDesign 的對應頁面上自動生成綠色的輔助定位方框。

**使用方法：**
1. 在 InDesign 中打開已置入漫畫圖檔的文檔。
2. 雙擊執行 `創建方框.jsx`。
3. 彈出視窗時，選擇對應的 JSON 檔案（若有主控腳本傳遞路徑則會自動讀取）。
4. 執行完畢後，腳本會在文檔最頂層建立一個名為 `[輔助定位] 完整矩形框` 的圖層，並自動鎖定。畫面上會出現與原對話框位置對應的綠色實線方框。

> **💡 手動對齊提示：** 若後續需要手動拖曳文字框對齊，建議關閉 `[檢視] > [靠齊文件格點]`，並勾選 `[偏好設定] > [智慧型參考線] > [對齊物件中心]`。拖曳時看到十字交叉線即代表中心對齊。

---

## 2. 選中文字中心對齊方框.jsx

**用途：** 
將選定的文字框（或物件），一鍵自動吸附（對齊）到剛才生成的綠色輔助方框的中心點。此版本經過極速優化，搜尋範圍僅限當前頁面，即使文檔有數百頁也能瞬間完成。

**使用方法：**
1. 確保已經先執行過 `創建方框.jsx`，畫面上已經存在綠色的輔助方框。
2. 在 InDesign 中選取你要對齊的文字框（支援一次選取多個）。
3. **⚠️ 關鍵前提：** 你選取的文字框，必須與目標綠色方框有**部分重疊（相交）**，腳本才能判斷你要對齊到哪一個框。
4. 雙擊執行 `選中文字中心對齊方框.jsx`。
5. 選中的文字框將會瞬間移動，使其中心點與相交的綠色方框中心點完美對齊。
